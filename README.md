# Random Password Generator (Генератор случайных паролей)
**Автор:** Богдан Набиев
**Вариант:** Random Password Generator
**Дата сдачи:** 04.05.2026
---
## Описание программы
Это удобная графическая программа для быстрого создания надежных паролей. Вы выбираете длину пароля и типы символов (цифры, буквы, спецсимволы). После нажатия кнопки программа сгенерирует пароль, который можно легко скопировать в буфер обмена. Также программа хранит последние 10 сгенерированных паролей в истории для быстрого доступа. Всё работает на Windows и сохраняет историю в файл.
